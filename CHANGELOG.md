# Changelog

Todas as mudancas importantes deste projeto serao documentadas neste arquivo.

## [2.0.0] - 2026-07-18

### ✨ Novidades

- **Sistema de Progresso**: Salva automaticamente licoes concluidas no navegador
- **Semana 3 Completa**: Estruturas condicionais (if, elif, else, and, or, not)
- **Indicadores Visuais**: Checkboxes nas licoes completas
- **Barra de Progresso**: Mostra percentual de conclusao por semana
- **Botao de conclusao**: Marque licoes como concluidas ao terminar

### 🎨 Melhorias de Design

- Checkboxes circulares nas licoes
- Animacoes suaves de progresso
- Melhor feedback visual para licoes completas
- Estilo atualizado do botao de conclusao

### ♿ Acessibilidade

- Adicionados ARIA labels em navegacao
- Roles semanticos (main, complementary, navigation)
- Atributos aria-selected para tabs
- Melhor suporte para leitores de tela

### 📱 Responsividade

- Layout adaptavel para mobile
- Sidebar colapsavel em telas pequenas
- Touch-friendly buttons

### 📚 Documentacao

- README atualizado com novas funcionalidades
- Guia de deploy para GitHub Pages (DEPLOY.md)
- Exemplos de como adicionar novas semanas

### 🔧 Tecnico

- Arquivo `.nojekyll` para GitHub Pages
- Meta tags para SEO e compartilhamento social
- Sistema modular de progresso (progress.js)
- LocalStorage para persistencia de dados
