## 🚀 Como Usar

### 1️⃣ Testar Localmente

```powershell
cd D:\projetos\python\study-site
python -m http.server 8000
```

Acesse: **http://localhost:8000**

### 2️⃣ Publicar no GitHub Pages

```powershell
git init
git add .
git commit -m "Site de estudos Python completo"
git remote add origin https://github.com/SEU-USUARIO/study-site.git
git push -u origin main
```

Depois:
- GitHub > Settings > Pages
- Source: main branch
- Aguarde 2-5 minutos

Site online: **https://SEU-USUARIO.github.io/study-site**
